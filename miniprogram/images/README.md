# 图标资源清单

请手动添加以下图标文件到 miniprogram/images 目录下：

1. score.png - 计分图标
2. score_active.png - 选中状态的计分图标
3. history.png - 历史图标
4. history_active.png - 选中状态的历史图标
5. profile.png - 个人图标
6. profile_active.png - 选中状态的个人图标
7. default-avatar.png - 默认头像图标
8. add-player.png - 添加选手图标

这些图标用于应用的导航栏和用户界面。推荐使用符合浅绿色主题的图标。
